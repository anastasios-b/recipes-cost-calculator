This example demonstrates how a simple recipie calculation works.

Let's say we have a product recipie for Metal Chair.

This product is built using some basic materials, including bolts, metal rods etc.

It also requires people to work on it (assembly & painting).

API Documentation

This server provides the following REST API endpoints for recipe management:

ðŸ“– Recipe Management

GET /api/recipes

Description: Retrieve all recipes

Response: Array of recipe objects

GET /api/recipes

GET /api/recipes/:id

Description: Retrieve a specific recipe by ID

Parameters: id (string) - Recipe ID

Response: Single recipe object

GET /api/recipes/123e4567-e89b-12d3-a456-426614174000

POST /api/recipes

Description: Create a new recipe

Body: Recipe object (see schema below)

Response: Created recipe with ID and timestamps

POST /api/recipes

PUT /api/recipes/:id

Description: Update an existing recipe

Parameters: id (string) - Recipe ID

Body: Updated recipe object

Response: Updated recipe object

PUT /api/recipes/123e4567-e89b-12d3-a456-426614174000

DELETE /api/recipes/:id

Description: Delete a recipe

Parameters: id (string) - Recipe ID

Response: 204 No Content

DELETE /api/recipes/123e4567-e89b-12d3-a456-426614174000

ðŸ’° Cost Calculation

GET /api/recipes/:id/cost

Description: Get detailed cost breakdown for a specific recipe

Parameters: id (string) - Recipe ID

Response: Detailed cost analysis including parts, labor, waste factor

GET /api/recipes/123e4567-e89b-12d3-a456-426614174000/cost

GET /api/recipes/cost/summary

Description: Get cost summary for all recipes

ðŸ”§ Usage Examples

Create a new recipe:

fetch('/api/recipes', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: "Wooden Table",
    weight: 25.0,
    dimensions: {
      length: 120,
      width: 80,
      height: 75,
      unit: "cm"
    },
    yield_percentage: 0.90,
    waste_factor: 0.10,
    unit_of_measure: "piece",
    inventory_location: "workshop",
    parts: [
      {
        name: "Wood Plank",
        quantity: 6,
        cost_per_unit: 45.00
      }
    ],
    labor: [
      {
        type: "Carpentry",
        hours_needed: 4,
        cost_per_hour: 35.00
      }
    ]
  })
});

Get recipe cost breakdown:

fetch('/api/recipes/recipe-id/cost')
  .then(response => response.json())
  .then(data => {
    console.log('Total cost:', data.cost_summary.total);
    console.log('Parts cost:', data.parts.total);
    console.log('Labor cost:', data.labor.total);
  });
